# HandbrakeMap-ATtiny85

Driver-less remapper for ATtiny85-based USB handbrake.  

## Lore
Got [this cheap handbrake](https://allegro.pl/oferta/hamulec-reczny-usb-hydrolapa-logitech-thrustmaster-12129762247), which apparently needs some "Digistump Digispark" drivers to work, aint no f way in hell i'm installing shi like that.  

* **HandbrakeMap.exe** - Maps `PRESS | RELEASE` actions to `VK_SPACE` (spacebar) by default.
* **HIDScan.exe** - Raw input data dumper
